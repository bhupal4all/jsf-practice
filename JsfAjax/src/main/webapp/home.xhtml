Java Code
@ManagedBean(name = "helloBean", eager = true)
public class HelloBean {
	String message;
	public String getMessage() {
		return message;
	}
	public void setMessage(String message) {
		this.message = message;
	}
}

HTML code
<h:inputText id="input" value="\#{helloBean.message}">
	<f:validateLength minimum="3" />
	<f:ajax event="keyup" render="inputerror echo" />
</h:inputText><h:message id="inputerror" for="input"/>
<br />

Entered Value: <h:outputText id="echo" value="\#{helloBean.message}" />

default
f:ajax execute default is @this
After entering minimum of 3 characters, entered value would be appeared.
