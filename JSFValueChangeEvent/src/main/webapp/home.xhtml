Value Change Event Practice

Value Change Event Listener Method

Java Code
@ManagedBean(name = "helloBean", eager = true)
public class HelloBean {
	public void menuValueChangeEvent(ValueChangeEvent event) {
		System.out.println("menuValueChangeEvent\n\tOld Value="
				+ event.getOldValue() + "\n\tNew Value=" + event.getNewValue());
	}
}

HTML Code
<h:selectOneMenu value="hash{helloBean.debugLevel}"
	valueChangeListener="hash{helloBean.menuValueChangeEvent}"
	onchange="submit()">
	<f:selectItem itemValue="All" />
</h:selectOneMenu>

Note: form needs to be submitted when value got changed
